There are three mailing lists for Glade:

  • glade-announce@ximian.com - A moderated list for announcements of new versions of Glade and other closely-related tools, such as libglade and the various code generators.

  • glade-users@ximian.com - For discussions about using Glade to build applications.

  • glade-devel@ximian.com - For discussions about the development of Glade itself.

To subscribe, unsubscribe, or view the archives of the lists go to the glade-announce, glade-users or glade-devel mailman web pages.

Alternatively you can use email to perform commands by mailing glade-announce-request@ximian.com, glade-users-request@ximian.com or glade-devel-request@ximian.com, with commands such as 'subscribe' and 'help' in the subject line or the message body.





An archive of the old glade-devel list, up to Aug 31 2000, can be found at:

  • http://rpmfind.net/tools/glade/messages/
