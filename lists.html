Glade Mailing Lists

There are two mailing lists for Glade:

  • glade-users@ximian.com - For discussions about using Glade to build applications.
  • glade-devel@ximian.com - For discussions about the development of Glade itself.

To subscribe, unsubscribe, or view the archives of the lists go to the glade-users or glade-devel mailman web pages.

Alternatively you can use email to perform commands by mailing glade-users-request@ximian.com or glade-devel-request@ximian.com, with commands such as 'subscribe' and 'help' in the subject line or the message body.

There is also a mailing list for the C++ code generator here:
