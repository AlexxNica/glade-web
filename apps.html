Glade Example Applications

Here are a few applications that have been created with Glade:

  • PHOEBE - used for modeling eclipsing binary stars. It has a very complicated interface, created in Glade (the largest Glade file I have ever seen!). Recent versions (0.30) use Glade 2, while previous versions used the original Glade.
  • glocate2 - a simple file locator application using Glade 2 and libglade2 (in Spanish).
  • gtkpod - a GUI for Apple's iPod.
  • gmodconfig - a tool that helps Linux users manage kernel modules.
  • Meld - a visual diff an merge tool.
  • gonvert - A unit conversion utility.
  • TiLP - a linking program for Texas Instruments' graphing calculators, using libglade.
  • gtk2edit - a text editor, slightly more complicated than the Glade example editor application.
  • Sussen - a security scanner for GNOME.
  • configure-thinkpad - an IBM ThinkPad configuration tool.
  • BoardGame Tool - a boardgame utility with a C++ front-end built with Glade and a Python back-end.
  • polyxmass - a computer framework for mass spectrometry, specifically for polymers.
  • gbarco - a utility to control a barco crt-projector over RS232.
  • Greyhounds - a greyhound racing and breeding game.
  • Bygfoot - a football manager game.
  • BooleanMinimizer - minimizes boolean functions; written in C++; runs on Linux, Solaris, Win32.
  • DTraq - an Ada 95 data logging and interactive review and playback tool.
  • GNOME-IOR - parses and displays CORBA object references, with ping-like features to check if objects are alive (used libglade).
  • RealmForge GDK - a cross-platform game development framework and toolkit written in Mono/C# and powered by the Axiom 3D engine. It will allow for the rapid development of cutting-edge script & data-driven games and MMORPGs.
  • gFLAMEd - a graphical editor for algorithms derived using FLAME.
  • configure-trackpoint - a graphical TrackPoint configuration tool.
  • Warehouse Manager - a business application designed to allow you to manage several aspects of your warehouse based enterprise from your computer.
  • keyTouch - makes it possible to easily configure the extra function keys (like multimedia keys) of your keyboard. You can define which program will be executed while pressing a key.
  • Xcave - a wine manager for linux users.
