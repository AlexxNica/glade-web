Applications


If you have created an application using Glade that would be a good example for others to look at, email the URL and a short description to damon at gnome.org. You can also use our Glade logo to link from your home page to Glade.

Glade 2 Example Applications

  • glocate2 - a simple file locator application using Glade 2 and libglade2 (in Spanish).

Glade 1 Applications

  • Progeny Debian - Progeny's commercial release of Debian makes extensive use of Glade. The installer and configuration system (a GNOME frontend to Debian's Debconf system) is built using Glade.
  • AlsaPlayer - a PCM (audio) player for Linux/ALSA.
  • Superficie - a tool for visualizing 3D surfaces.
  • XFce - a window manager/desktop.
  • Lycadican - Role-Playing Game.
  • gtktetcolor - A Tetris-type game.
  • Beat Monkey - Simple calculator for pitch shifting audio samples.
  • GCO (GNOME Comics Organizer) - a database for keeping track of your comic collection.
  • GnOpenGL3ds - a 3ds file viewer. It requires OpenGL(or Mesa), gtkgl and Gnome.
  • GuiSlp - a graphical package manager for stampede linux packages.
  • Tapelab - a Gnome program to write and print audio cassette labels.
  • Gnorponocal - a fully-featured, scientific RPN keyboard based calculator for Gnome.
  • Fire Gnome - a firewalling toolkit for ipchains (initially). In early stages of development.
  • Xcave - a wine manager for linux users.
  • GtkBrew - a mini-suite of applications for Homebrewing. (Only in Italian at present.)
  • Gnome File Viewer - a convenience tool for scanning directories and viewing the contents of the files within them.
  • DPS-FTP - a Gnome ftp client.
  • ToutDoux - a Project Manager.
  • Gnome ID3 to HTML - generates a HTML sheet for selecting MP3s, by extracting their Id3 tags.
  • Sodipodi - a vector drawing application for GNOME.
  • gMGAclock - Matrox G400 tweaking utility.
  • Rubrica Italiana - a telephone card application in Italian. This is also available at Roberto A. Foglietta's home page.
  • NumExp - a mathematical computation environment and programming language which uses libglade.
  • g2100config - a tool to configure and use HP LaserJet 2100 series printers.
  • pfim (points from image) - digitizes data points from an image.
  • gato - an interface to the UNIX 'at' command.
  • gcrontab - A crontab editor.
  • Dr Genius - Dr Genius is an interactive geometry program plus a calculator similiar in some aspects to bc and Matlab.
  • gHW - an interface to the /proc filesystem.
  • Povray texture editor - a texture editor for povray 3.1 (in Spanish).
  • XRolyPoly - a rolodex app to store phone numbers, addresses etc.
  • George - an MP3 file reorganizer and CD creator (using Glade-Perl).
  • Chasm - a cave survey visualisation tool.
  • gtk+licq - a GTK+ plugin for licq.
  • g3DGMV - a free viewer for Digital Elevation Model (DEM) and Digital Line Graphs (DLG) maps. The maps are rendered as 3D images that can be manipulated by the user and viewed from different angles.
  • ptf - a front-end to the network utilities ping, traceroute and finger.
  • Local Resources Database for Eranakulam District in India - EIID (a charitable society resulting from the Peoples Plan Campaign) has developed a Local Resource Database to strengthen micro-level planning, using LINUX, PostGreSQL, GTK+ and Glade. Currently there is no web page, though they can be reached via email at eiidp@md3.vsnl.net.in
  • hOpla - links between XML files and a database (postgreSQL).
  • ZoomEdit - a GIS-like program for viewing and editing NOAA hydrographic sonar data.
  • gxhost - an interface to the xhost command.
  • Figaro's Password Manager - a GNOME application for storing and using passwords.
  • Gnome UPS Monitor - Monitors battery, input current, load etc. on a UPS.
  • Gm3u - an MP3 play-list editor for Gnome.
  • XRoadMaker - a Gnome PCB designer with a customizable router engine.
  • GnoZip - a zip archiver.
  • gRustibus - a Gnome frontend to M.A.M.E. (Multi Arcade Machine Emulator).
  • XAmoba - a networked five-in-a-row/noughts and crosses game.
  • Gabber - a GNOME Jabber client.
  • Gnomerar - a frontend for the linux rar archiver.
  • gAgenda - saves diary entries and export them to a HTML page.
  • GTelnet - a GTK+ telnet client.
  • GTKstereograph - a stereogam generator.
  • Tinlib - a Gnu C library for creating Triangulated Irregular Networks (TIN meshes) from raw XYZ data. As a bonus, it also creates a 'shrink wrapped' 2d vector outline of the data's boundaries.
  • xconvers - a network client for amateur radio convers (port 3600).
  • simple client/server app - a simple client-server communication system written by 4 Italian students (in Italian).
  • Goblin - an IDE for Web site development and local/remote management.
  • Global3D - a polygonal modeler under linux and irix.
  • GpetView - a medical image (ANALYZE format image) viewer.
  • f2e - a simple app to convert between French francs and Euros.
  • gfslicer - a utility to slice (split) and deslice (join) your files.
  • GDiskImage - stores images of floppy disks, which is very convenient for system disks, small OS'es and so on. It uses PostgreSQL to maintain its archives.
  • Gcomm - a GTK+ communications program similar to seyon.
  • DiaCatalog - Slides/Photos cataloging program for speleologic groups (italian only, for now).
  • gnews - scrolling news ticker applet for the gnome panel, displays headlines from known news sites. (The dialog skeletons were built with glade.)
  • pybber - a Jabber client (Instant Messaging) using Glade and PyGNOME/PyGTK.
  • NV-Tuner - a tweaking utility for the Nvidia drivers.
  • gWine - a GUI frontend to WINE, that lets you run Windows apps without using the command line.
  • Gyach - A Yahoo! Chat client.
  • MidiMountain - a complete Midi sequencer aimed at editing standard Midi files.
  • Siphon - a GTK+ SIP Soft Phone.
  • Jikanda - a simple timer.
  • xglobeconfig - a GUI front end for xglobe.
  • Fanorona - a traditional game from Madagascar. In English, French, German, Japanese, and Spanish.
  • Coriander - a GUI for controlling a Digital Camera over the IEEE1394 bus.
  • Dilutions - a simple application for buffer preparation in chemistry/biochemistry (to help you calculate dilutions etc.).
  • Gquotes - a Gnome application for retrieving stock quotes from many web servers. (The author describes it as a good example of the use of Glade & libglade.)
  • g-page - an application designed to send text messages to pagers or PCS phones.
  • Oww - runs a Dallas Semiconductor 1-wire weather station.
  • Secure Password Generator - allows a person to create a variable length password depending on the options they select.
  • gvlc (the Gnome VideoLAN Client) - an MPEG and DVD player for Unix.
  • ggnuplot - small graphical interface for the unix graphing program 'gnuplot'. Allows quick and dirty graphs of data files to be written to encapsulated postscript, with an optional latex container file.
  • gdcalc - a GLADE/GTK+ Reverse Polish Notation (RPN - HP-like) and algebraic (conventional = Texas/Casio-like) calculator for Unix and Linux with stats, financial, programming and scientific functions.
  • Rubrica - a simple address book application (in Italian, I think).
  • Netuse - Squid log graphical output. It shows a bar time for every IP using Squid proxy for surfing internet and draws a line at every minute user makes "click" in his browser.
  • gonvert - A unit conversion utility.
  • grocget - a grocery shopping helper.
  • tvnow - a TV listing viewer for the TVNOW subscription service.
  • gsensor - a graphical interface to the lm_sensors hardware monitoring package.
  • Intra Chat - a multi-platform GUI Chat facility for both Linux and Microsoft Windows machines in an office or institutional Intranet environment. It Features Chat room, Instantant Messaging, FTP and a White board. It also includes a dedicated chat server with very mall foot print.
  • Exult Studio - a map-editor for Exult, the open-source game engine for playing Ultima7.
  • gssh - a frontend to ssh made to look like kssh. (The link is to the tarball.)
  • Xccf - creates italian fiscale code (in italian).
  • GCTB (GNU Contabilidad) - a basic account system with licence GNU GPL (in Spanish, I think).
  • gechoserver - a graphic echo server.
  • Crank - a cryptanalysis toolkit.
  • SimpleCDR-X - a tool for cd burning, mastering, ripping and audio compression.
  • dSQL - a simple multi-database query tool just like Microsoft Query Analizer. Uses Gtk-PERL.
  • Terraform - an interactive terrain/height_field editor.
  • PPPMON - an RP3 clone. It shows the activity in the PPP connection, and starts/stops the PPP connection in an easy way.
  • gmp3burn - a frontend to mp3burn perl script. It can do mp3 lists manipulation and id3tags reading and has a very friendy gui.
  • ROME Target Builder - a build configuration tool for the ROME embedded operating system.
  • GCronTime - a tool for editing crontab files.
  • BookBase - a MySQL database application for book collectors.
  • GnomeZine - a database manager for magazines articles, currently using MySQL.
  • AVR Delay loop Generator - a simple app to create delay loops in assembler for AVR RISC processors. The users just specifies the time to delay.
  • gEuCo - the gnome Euro Conversor. It converts the currencies from the Euro countries to Euros and vice-versa.
  • Gneuro - another euro/currency converter. This one has a panel applet mode as well.
  • polyXmass - a scientific program aimed at simulating/analyzing mass spectrometric data on a variety of polymers.
  • XMMS-Crossfade - A Crossfading/Gapless Plugin for XMMS.
  • HushTones IM - an IM client for Unix based platforms, packed with unmatched security features such as RSA Public Key encryption and IDEA encryption.
  • Stoq - retail business management software (GPL'ed).
  • GnomeRSVP - a Rapid Serial Visual Projector for computer assisted speed reading.
  • Knocker - a simple, versatile, and easy-to-use TCP security port scanner written in C.
  • GnomeLDAP - a tool for modeling and administration of LDAP.
  • Bond - a library that links Glade XML files to PostgreSQL databases and makes the application data-aware. This is great if you are writing a database application but don't want to have to write C code for all your widgets or deal with messy SQL statements.
  • OggDoctor - a toolkit for music in ogg/vorbis file format.
  • Xsox - graphical X front end for the Sox sound program.
  • Schlange - An OpenGL 3D snake-clone; multiplayer-capable.
  • SIPSet - a SIP (Session Initiation Protocol) based softphone. (Also runs on an iPAQ running Linux with 802 wireless access.)
  • TiLP - a linking program for Texas Instruments' graphing calculators.
  • pxesconfig - the graphical configuration of PXES Linux Thin Client. (PXES is a micro Linux distribution for thin clients.)
  • gwaferprog - a GUI for programming Gold Wafer cards and Silver cards.
  • Gwine - a wine cellar manager written in Perl (with Glade-Perl).
  • Gspoof - allows you to create and send arbitrary TCP/IP packets.
  • Gnopa - a frontend for interfacing with the Compaq PA1 and PA2 Personal Audio MP3 players.
  • gjots - organises your jottings and musings in a tree structure, and can also keep them encrypted if ccrypt(1) is installed.
  • Meld - a diff and merge tool.
