Glade Download Page

Please report bugs for the new glade on bugzilla.gnome.org, with the product set to 'glade3'.

Source Code:

  • Glade 3.0 (Aug 11 2006, 1 MB)

    This is the first full featured release series of the new glade tool, it depends on GTK+ 2.8 and will allow the user to include properties, signals and widgets from GTK+ >= 2.8.

  • Glade 2.10.0 (Mar 16 2005, 3.7 MB)

    This version of Glade is for GTK+ 2.6 and GNOME 2.10. Note that interfaces created with Glade 2.10.0 will need GTK+ 2.6 and GNOME 2.10.

  • Glade 2.6.8 (Dec 21 2004, 3.7 MB)

    This version of Glade is for GTK+ 2.4 and GNOME 2.6/2.8. Note that interfaces created with Glade 2.6.0 will need GTK+ 2.4 and GNOME 2.6.

  • Glade 2.0.1 (Nov 7 2003, 3.0 MB)

    This version of Glade is for GTK+ 2.0/2.2 and GNOME 2.0/2.2/2.4.

  • Glade 0.6.4 (Feb 1 2002, 2.6 MB)

    This version of Glade is for the older GTK+ 1.2 and GNOME 1.4 platform. It is not supported any more.
